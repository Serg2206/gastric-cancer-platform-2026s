## 🗺️ Дорожная карта курса (12 Интерактивных Этюдов)

### 📘 Блок I: Эпидемиология и Смена Парадигмы
* **Модуль 1. ЭТЮД I: «Демографический шизофрен»**
    - *Тема:* Рак молодеет, хирург стареет. Эпидемиология YOGC. Мутации *CDH1*.
    - *Интерактив:* Ретроспективный клинический аудит центра по чек-листу.
* **Модуль 2. ЭТЮД II: «Мультиомиксная навигация: от ctDNA к скальпелю»**
    - *Тема:* Жидкостная биопсия (CAPP-seq) и ПЭТ/КТ с ⁶⁸Ga-FAPI против микроскопического карциноматоза.
    - *Интерактив:* Ветвящийся кейс пациентки 42 лет (изменение стратегии cT2N1 -> cT4aN2).

### 📗 Блок II: Периоперационные Алгоритмы и Органосохранение
* **Модуль 3. ЭТЮД III: «Протокол MATTERHORN и Новая Эра Иммунотерапии»**
    - *Тема:* Чекпойнт-ингибиторы (Дурвалумаб/Пембролизумаб), стратификация по MSI-H/dMMR и PD-L1 CPS.
* **Модуль 4. ЭТЮД IV: «CRITICS-II и точка возврата»**
    - *Тема:* Тотальная неоадъювантная терапия (TNT) и концепция *Sterilized Field* (фиброз вместо опухоли).
* **Модуль 5. ЭТЮД VII: «Проксимальная резекция с DTR»**
    - *Тема:* Органосохранение при аденокарциноме GEJ Siewert II. Реконструкция Double Tract Reconstruction.

### 📙 Блок III: Прецизионная Робототехника и Визуализация
* **Модуль 6. ЭТЮД V: «Da Vinci 5: Сенсорика против слепоты»**
    - *Тема:* Force Sensing (тактильная обратная связь 0,1–10 Н), AI-сегментация границ и AR-навигация при диссекции по Киму.
* **Модуль 7. ЭТЮД VI: «ICG-Флуоресценция: Зелёный свет в тумане войны»**
    - *Тема:* Лимфокартирование и количественная оценка перфузии анастомоза технологией FLOW 800.
* **Модуль 8. ЭТЮД VIII: «Аутермост-слойный доступ (OLA) при cT4a»**
    - *Тема:* Бескровные пространства (Avascular Plane) по данным RCT UMC-UPPERGI-01 (2025).

### 📕 Блок IV: Запущенные Стадии, Осложнения и Командная Мудрость
* **Модуль 9. ЭТЮД IX: «PIPAC и HIPEC: Химия под давлением»**
    - *Тема:* Аэрозольная химиотерапия при перитонеальном карциноматозе. Конверсия в CRS.
* **Модуль 10. ЭТЮД XI: «MDT 2.0: От совещания к коллективному разуму»**
    - *Тема:* Интеграция AI-решений (Watson for Oncology) и FHIR-протоколов в мультидисциплинарные консилиумы.
* **Модуль 11. ЭТЮД XII: «Памоморфоз как пророчество»**
    - *Тема:* Шкала Mandard TRG, irPR и TILs. Мониторинг MRD (Signatera) на 7 и 30 дни после операции.
* **Модуль 12. ЭТЮД X & Эпилог: «Симуляционная революция и Манифест Хирурга»**
    - *Тема:* Пятиуровневый симуляционный лабиринт (VR -> Swine Lab -> Thiel-кадаверы -> Цифровые двойники).

## 🛠️ Технологический стек платформы

- **Frontend:** Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn UI.
- **Backend & База данных:** Supabase (PostgreSQL), Clerk Auth (авторизация врачей).
- **AI-ядро:** Anthropic Claude 3.5 Sonnet API (потоковый стриминг ответов для AI Tutor).
- **Платёжный шлюз:** Stripe (B2C-подписки и B2B/институциональные лицензии).

## 🚀 Быстрый старт для разработчиков

### 1. Клонировать репозиторий и установить зависимости:
```bash
git clone https://github.com/Serg2206/gastric-cancer-platform-2026.git
cd gastric-cancer-platform-2026
npm install
```

### 2. Настроить переменные окружения:
Создайте `.env.local` в корне проекта на основе шаблона:

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_key
CLERK_SECRET_KEY=your_clerk_secret
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
ANTHROPIC_API_KEY=your_claude_api_key
STRIPE_SECRET_KEY=your_stripe_key
```

### 3. Запустить локальный сервер:
```bash
npm run dev
```
Приложение будет доступно по адресу http://localhost:3000
